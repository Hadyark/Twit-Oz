**Structure des fichiers**
* **Enoncé.pdf:** Contient l'énoncé du projet
* **Groupe BB - TwitOz rapport.docx:** Contient le rapport du projet au format docx
* **Groupe BB - TwitOz rapport.pdf:** Contient le rapport du projet au format pdf
* **Makefile:** Contient un ensemble de directives pour exécuter/compiler le projet
* **Makefile.bat:** Ccompile et lance le MakeFile
* **Parser.oz:** Contient le code du Parser (crée un dictionnaire spécifique à un fichier à partir des phrases envoyées par le Reader)
* **Reader.oz:** Contient le code du Reader (lit les fichiers ".txt" et les envoie au Parser)
* **Saver.oz:** Contient le code du Saver (merge tous les dictionnaires envoyés par les Parser en un seul dictionnaire)
* **main.oz:** Contient le code pour lancer l'application et le GUI
* **tweets** (dossier): Un dossier contenant tous les fichiers utilisées par l'application
